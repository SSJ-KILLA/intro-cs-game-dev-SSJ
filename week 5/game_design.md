# Game Title
Asteroid Escape

## Game Genre
Top-down dodger

## Core Gameplay Loop
The player moves a ship around the screen to avoid incoming asteroids. As time passes, asteroids spawn faster, increasing difficulty. The player repeats dodging and repositioning to survive as long as possible.

## Player Actions
- Move up
- Move down
- Move left
- Move right
- Dodge
- Restart

## Win Condition
The player wins by surviving for a set amount of time (60 seconds).

## Loss Condition
The player loses if their ship collides with an asteroid.

## Scope Statement
This game will have one screen, one type of obstacle, no shooting, no power-ups, and simple shapes instead of detailed graphics.
